# Multithreaded Port Scanner

A Java-based network security tool that scans a target host
to identify open ports and active services.

## Features

- TCP port scanning
- Multithreaded scanning
- Service detection
- Fast scanning with timeout handling

## Technologies

Java
Socket Programming
Multithreading
TCP/IP Networking

## Usage

Enter target IP and the program scans ports 1–1024.

## Example Output

Port 22 → OPEN (SSH)
Port 80 → OPEN (HTTP)
Port 443 → OPEN (HTTPS)
